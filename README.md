# ChoiceMensWear

A premium full-stack menswear storefront concept built with Next.js.

## Features

- Editorial luxury homepage inspired by premium fashion retail patterns
- Dynamic collection switching with curated product cards
- Full-stack API routes for catalog data and concierge appointment requests
- Responsive layout for desktop and mobile

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
